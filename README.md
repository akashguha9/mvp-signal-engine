\# MVP Signal Engine



A multi-layer signal system combining:



\- News (GDELT)

\- Prediction markets (Polymarket, Kalshi)

\- Global index data

\- Time-series signal generation

\- Backtesting engine



\## Pipeline



Signal Flow:



News → Events → Index Mapping → Timeseries → Signals → Backtest



\## Outputs



\- output\_timeseries\_dataset.csv

\- output\_signals.csv

\- output\_backtest\_summary.csv
